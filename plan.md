# Bow & Crescent Website Plan

## Context
A professional website for **Bow & Crescent Single Member PC** (Μονοπρόσωπη ΙΚΕ), a Greek IKE providing US and international tax consulting for private equity in the digital asset space. The site satisfies Greek legal requirements for IKE transparency while presenting a credible, modern face to clients.

## Tech Stack
- **Plain HTML + CSS** — no frameworks, no build tools, no JavaScript dependencies
- Single `index.html` with anchor-linked sections
- Inline SVG logo (no external image dependencies)
- Responsive design via CSS media queries

## Color Scheme: Navy + Gold
| Role       | Color                    |
|------------|--------------------------|
| Background | `#0a1628` (deep navy)    |
| Surface    | `#111f36` (lighter navy) |
| Text       | `#f0f0f0` (off-white)    |
| Accent     | `#c9a84c` (warm gold)    |
| Muted text | `#8899aa` (slate)        |

## Site Structure (single page)
1. **Header / Navigation** — Logo + company name, nav links (About, Services, Contact)
2. **Hero Section** — Company name, tagline, one-liner
3. **About Section** — Firm background, expertise, focus areas
4. **Services Section** — Four service cards with descriptions
5. **Contact Section** — Email, professional CTA
6. **Footer** — Greek IKE legal disclosures, copyright
